# AccessPoint Dx Website Prototype

AccessPoint Dx is a Minnesota-centered laboratory services platform organized around three clear diagnostic pathways:

1. Provider Partnerships
2. Axis Point Premium™ Direct-Access Testing
3. Occupational & Compliance Screening

This repository contains the static website prototype for the AccessPoint Dx platform. It is built with HTML, Tailwind CSS CDN, and vanilla JavaScript for simple prototype interactions.

## Current Pages

- `index.html`  
  Main AccessPoint Dx homepage and three-path diagnostic gateway.

- `provider-partnerships.html`  
  Provider-first laboratory partnership page for clinics, specialty practices, long-term care, and care organizations.

- `direct-access-testing.html`  
  Axis Point Premium™ direct-access testing page with self-pay wellness testing flow.

- `testing-profile.html`  
  Prototype profile/status page for direct-access testing orders.

- `occupational-compliance-screening.html`  
  Occupational and compliance screening pathway for employers, workforce programs, and public-sector organizations.

- `privacy-policy.html`  
  Draft privacy policy page.

- `terms.html`  
  Draft terms of use page.

- `refund-policy.html`  
  Draft refund policy page for direct-access testing workflows.

- `hipaa-notice.html`  
  Draft HIPAA notice / notice of privacy practices page.

## Project Notes

This is a static front-end prototype. It does not currently include a production backend, live payment processor, real authentication system, live laboratory information system integration, or production PHI workflow.

Some prototype interactions may use browser `localStorage` to simulate order creation and profile handoff. This is for demonstration only and should be replaced with secure backend infrastructure before production use.

## Brand Positioning

AccessPoint Dx is positioned as a local Minnesota diagnostic partner with clear boundaries between:

- Provider-directed clinical laboratory partnerships
- Consumer direct-access wellness testing
- Occupational and compliance screening workflows

The platform is designed to support local care teams, not replace provider relationships.

## Legal Notice

The privacy policy, terms, refund policy, and HIPAA notice included in this repository are draft prototype language. They should be reviewed by qualified legal and healthcare privacy counsel before production launch.
